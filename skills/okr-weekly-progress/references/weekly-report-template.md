# Weekly Report Template

Use this template when the user wants a paste-ready weekly report.

## Chinese compact format

### 本周工作概述

- 用 2 到 4 句话概括本周核心推进事项。

### 对照 OKR 的工作进展

#### Objective / KR 1: [名称]

- 本周完成：
- 产出/结果：
- 当前状态：

#### Objective / KR 2: [名称]

- 本周完成：
- 产出/结果：
- 当前状态：

### 工作分类

- 调研分析：
- 方案设计：
- 开发实现：
- 测试验证：
- 文档汇报：
- 协同沟通：

### 风险与问题

- 没有则明确写“暂无明显阻塞”。

### 下周计划

- 列 2 到 4 条，并尽量继续对应 OKR。

## Mentor/advisor format

### Weekly Summary

- Summarize the week's main progress in 2-3 sentences.

### Progress Against OKR

- Objective/KR:
- Completed this week:
- Evidence or deliverable:
- Status:

### Issues and Risks

- Note blockers, dependencies, or validation still needed.

### Plan for Next Week

- List concrete next actions tied to the same OKR where possible.
